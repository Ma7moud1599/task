# Task Manager — JavaScript

A lightweight task management app built with vanilla JavaScript. Organize tasks, set priorities, and track completion — all in the browser.

## Features

- Create tasks with title and optional description
- Mark tasks complete / incomplete
- Delete tasks
- Filter by status (all / active / completed)
- localStorage persistence

## Running

Open `index.html` in any browser — no server or build step required.

## Tech Stack

Vanilla JavaScript, HTML5, CSS3, localStorage
